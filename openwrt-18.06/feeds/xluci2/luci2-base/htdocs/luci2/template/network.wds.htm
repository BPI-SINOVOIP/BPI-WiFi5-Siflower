<%:STEP1: Go to Network-Scan to see which WIFI you want to connect. Remember ssid, bssid, channel and encryption%>

<%:STEP2: Add wwan in Network-Interface. protocol is DHCP client; Device in PhysicalSetting is sfix(sfi0 for 2.4G and sfi1 for 5G)%>

<%:STEP3: Add interface in Network-Wireless, ifname is sfi0(2.4G) or sfi1(5G), network choose wwan. Input ssid, bssid. Choose right channel and encryption(input Passphrase if needed)%>

<%:STEP4: Continue to refresh this page. You will see ip if connected. Then push the Enable button and WDS is finished%>

<%:STEP5: Click the Disable button and delete wwan, sfix to cancel WDS%>
